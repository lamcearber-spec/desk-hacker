## Review Categories (per app)

### A. Functionality
- [ ] Core features working?

- [ ] All user flows complete?

### B. Quality
- [ ] Tests exist and pass?
- [ ] Known bugs?
- [ ] Performance issues?
- [ ] Code quality/tech debt?

### C. UX/Design
- [ ] UI polished?
- [ ] Onboarding clear?
- [ ] Confusing flows?
- [ ] Mobile responsiveness?

### D. Revenue Readiness
- [ ] Pricing implemented?
- [ ] Payment integration?
- [ ] Subscription/billing logic?
- [ ] Free tier/trial defined?

### E. Launch Readiness
- [ ] App store listing ready? (mobile)
- [ ] Landing page? (web)
- [ ] Legal (privacy policy, terms)?
- [ ] Analytics/tracking?

### F. Distribution
- [ ] SEO strategy?
- [ ] Marketing channels identified?
- [ ] Launch plan?
- [ ] Target audience defined?
