# Amazon Associates Application Steps for DeskHacker

## Prerequisites
- Website live: https://lamcearber-spec.github.io/desk-hacker/
- 3+ original articles published (SEO-optimized reviews)
- Traffic starting (optional but helps approval)
- Tax info ready (US EIN/ITIN or non-US form W-8BEN)

## Steps
1. **Sign Up:** Go to https://affiliate-program.amazon.com/ → Create Account (use business email: info@desk-hacker.com or personal)

2. **Enter Website:** https://lamcearber-spec.github.io/desk-hacker/

3. **Describe Content:**
   - Niche: Home office furniture reviews
   - Traffic sources: SEO, social
   - Monetization: Affiliate links to Amazon products
   - Content plan: Weekly standing desk comparisons/reviews

4. **Tax Information:**
   - US: SSN/EIN
   - Non-US: W-8BEN form (no tax on earnings under thresholds)

5. **Phone Verification:** Receive code via call/text

6. **Submit & Wait:** Approval 1-7 days (sometimes longer). Check email.

## Post-Approval
- Install SiteStripe browser extension
- Generate links: Product → Get Link
- Track earnings in dashboard
- Comply: Disclose affiliates, no cloaking

## Tips for Approval
- Publish 5+ articles first
- Add privacy policy, about page
- Some organic traffic
- Avoid: Copied content, adult niches
