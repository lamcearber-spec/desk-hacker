## Site Outage & Fix (16:42-17:05 UTC)

### Issue: datev-bereit.de unreachable from outside
- Arber reported site down from home AND office
- DNS resolved correctly (Google, Cloudflare, Quad9 all returned correct IP)
- Hetzner DNS was timing out (separate issue, not root cause)

### Root Cause: Tailscale exit node left enabled
- Exit node was active from Reddit session days ago
- Routed ALL outbound traffic through Arber's PC
- Response packets went out wrong IP → clients rejected → timeout
- **Site was unreachable for potentially days without anyone noticing**

### Fix Applied:
1. Disabled exit node: `tailscale up --exit-node= --accept-routes`
2. Site immediately accessible

### Prevention Added:
1. Added exit node check to HEARTBEAT.md (auto-disable if found)
2. Added "PRODUCTION SERVER RULE" to AGENTS.md
3. Documented in TOOLS.md with warning
4. Logged to self-review.md as MISS

### Also Fixed:

- Hetzner DNS was unreliable, now have fallbacks
