# 🏛️ The Vault

Central knowledge base. Everything goes here.

## Structure
- **daily/** — Daily notes (YYYY-MM-DD.md)
- **projects/** — One folder per project (summary.md + items.json)
- **people/** — People we work with
- **companies/** — Company entities
- **inbox/** — Quick captures, unsorted notes
- **templates/** — Reusable templates
- **assets/** — Files, images, attachments

## Rules
1. Everything gets a home — no loose files in root
2. Daily notes go in `daily/`
3. New info → inbox first, sort later
4. Link between notes using `[[wikilinks]]`
